# AgencyAdminAPI

API para plataforma/site de agencia de veiculos. Tem por proposito: servir ao site da agencia de veiculos e painel admininstrador do site, permitindo gerenciar os veiculos disponiveis no site para venda. Centraliza a lógica de negócio tanto para a vitrine digital quanto para o back-office, oferecendo endpoints especializados na gestão de estoque, mídia de veículos e fluxos administrativos.
